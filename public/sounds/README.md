# Sound Files

Placer dine lydfiler i denne mappe for at aktivere baggrundsmusik i dynevælgeren.

## Påkrævede lyde

Følgende lydfiler skal være til stede for at systemet virker:

### Stemning valg
- `wind.mp3` - Kølig brise (blæsende vind)
- `nature.mp3` - Naturlig balance (naturlige lyde)
- `soft-rain.mp3` - Blød komfort (blid regn)
- `fireplace.mp3` - Hyggelig varme (knitrende ild)

### Temperatur valg
- `winter.mp3` - Har det koldt (vinter atmosfære)
- `ambient.mp3` - Helt tilpas (neutral ambient)
- `summer-night.mp3` - Har det varmt (sommernats lyde)

### Sæson valg
- `summer-night.mp3` - Sommer (genbruger fil)
- `ambient.mp3` - Hele året (genbruger fil)
- `winter.mp3` - Vinter (genbruger fil)

## Format
- Format: MP3
- Lydstyrke: Lydene afspilles ved 30% styrke for at være subtile
- Loop: Alle lyde loopes automatisk
- Filstørrelse: Hold filerne under 5MB for hurtig load
